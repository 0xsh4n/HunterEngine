## рЯФН What is HunterEngine?

HunterEngine is an **all-in-one automated bug bounty engine** that performs full-pipeline reconnaissance, vulnerability detection, and AI-powered triage вАФ from subdomain enumeration to report generation.

### вЬ® Key Features

| Feature | Description |
|---------|-------------|
| рЯХЈпЄП **Integrated Browser Auto-Crawl** | OWASP ZAPвАУstyle visible browser that auto-navigates targets: clicks links, fills forms, intercepts XHR/fetch/WebSocket traffic, discovers SPA routes |
| рЯФО **Full Recon Pipeline** | Subdomain enumeration вЖТ DNS resolution вЖТ live probing вЖТ tech fingerprinting вЖТ historical URL collection |
| рЯЫ°пЄП **15 Detection Modules** | XSS, CORS, SSRF, IDOR, JWT, Prototype Pollution, GraphQL, Auth flaws, Open Redirect, CSP, Subdomain Takeover, Race Conditions, Secrets, Dependencies, Crypto |
| рЯ§Ц **Local AI Triage** | Ollama / OpenAI-compatible LLM reviews findings, adjusts confidence scores, writes report summaries, and suggests safe validation steps |
| рЯФЧ **Vulnerability Chaining** | Automatically chains weak signals into higher-severity composite findings (e.g., Open Redirect + XSS вЖТ P1) |
| рЯМР **Embedded Proxy** | mitmproxy-based intercepting proxy with request history, replay, and passive analysis |
| вЪ° **Adaptive Rate Limiting** | Token-bucket rate limiter with automatic WAF/429 backoff |
| рЯЫ°пЄП **WAF Bypass** | User-agent rotation, referrer spoofing, TLS fingerprint impersonation |
| рЯУК **Multi-Format Reports** | Markdown, HTML, HackerOne, and Bugcrowd formatted reports |
| рЯТЊ **Pattern Memory** | SQLite-backed cross-session pattern storage for trend detection |

## рЯХЈпЄП Integrated Browser Auto-Crawl

HunterEngine v2.0.0 introduces **OWASP ZAPвАУstyle integrated browser crawling**. A visible Chromium window opens and autonomously explores the target while you watch:

### What It Does

- **рЯФЧ Clicks every link** вАФ `<a>` tags, navigation menus, breadcrumbs, sidebar links
- **рЯЦ±пЄП Clicks interactive elements** вАФ buttons, tabs, accordions, dropdowns, modals
- **рЯУЭ Auto-fills forms** вАФ intelligently fills email, name, search, password fields and submits
- **рЯМР Intercepts network traffic** вАФ captures every XHR, fetch, and WebSocket request as discovered endpoints
- **вЪ° Detects SPA routes** вАФ tracks `pushState`/`replaceState`, extracts React Router / Vue Router paths
- **рЯСБпЄП Watches DOM mutations** вАФ detects dynamically injected content
- **рЯУЄ Screenshots** вАФ captures evidence on each new page
- **рЯЫ°пЄП Scope-enforced** вАФ never navigates outside your defined boundaries

### Usage

```bash

python main.py crawl https://target.com

# With options

  --max-pages 300 \
  --max-depth 8 \

python main.py crawl https://target.com --headless
```

## вЪЩпЄП Configuration

### `config/scope.yaml` вАФ Target Scope

```yaml
program:
  name: "target-program"
  platform: "hackerone"         # hackerone | bugcrowd | yeswehack | private

in_scope:
  domains:
    - "*.target.com"
  cidrs: []                     # e.g., "10.0.0.0/8"

out_of_scope:
  domains:
    - "admin.target.com"
  keywords:
    - "delete"
    - "password_reset"

auth:
  type: "none"                  # none | bearer | cookie | basic
  token: ""
  cookie: ""
```
